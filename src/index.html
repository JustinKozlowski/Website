MAC-Swap-Extension

While working for UB ITCS, I would constantly have to use a number of DHCP management tools, often involving inputting a MAC address for the devices. Some tools used "-" as standard for the MAC address, while others use ":". This is a chrome extension that replaces every ":" with "-" and vice versa, for a given text input where the cursor is located. By doing this I wouldn't have to tediously replace each symbol for each tool, and had a faster work flow.
